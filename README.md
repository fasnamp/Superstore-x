# 📊 Superstore Sales Analysis Dashboard

An interactive **Sales Performance Dashboard** built using Microsoft Power BI to analyze sales, profit, orders, and product performance from the Superstore dataset.

The dashboard transforms raw sales data into clear visual insights that help understand business performance, product demand, and regional trends.

---

# 🎯 Project Objectives

- Analyze overall **sales performance and profitability**
- Compare **sales trends over time**
- Evaluate **product category and sub-category performance**
- Identify **top performing regions and segments**
- Measure **profit margins and discount impact**

---

# 📂 Dataset

The project uses the **Superstore dataset** containing transactional sales data.

### Main Tables

#### Orders
- Order ID  
- Order Date  
- Customer ID  
- Customer Name  
- Product Name  
- Category  
- Sub-Category  
- Region  
- Segment  
- Sales  
- Profit  
- Quantity  
- Discount  
- Shipping Days  

#### People
- Region  
- Person (Regional manager)

#### Returns
- Order ID  
- Returned status  

---

# 📊 Key Measures Created

The following DAX measures were created in Power BI:

- **Total Sales**
- **Total Profit**
- **Total Quantity Sold**
- **Orders Count**
- **Average Discount**
- **Average Shipping Days**
- **Profit Margin %**
- **Profit YTD**
- **Sales YoY Growth %**
- **Sales YTD**

These measures help evaluate financial performance and sales trends.

---

# 📈 Dashboard Insights

The dashboard provides insights such as:

- Sales performance across **regions and segments**
- Comparison of **category and sub-category sales**
- Identification of **high-profit and low-profit products**
- Impact of **discounts on profitability**
- **Year-to-date and year-over-year sales growth**
- Shipping efficiency through **average shipping days**
